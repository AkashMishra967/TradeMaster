TradeMaster is a full-stack stock trading web application inspired by Zerodha.
It replicates the core functionalities of a modern trading platform, allowing users to explore stock data, manage their investments, and simulate real-time trading in an intuitive environment.

This project demonstrates strong full-stack development skills including frontend design, backend architecture, API handling, and database management.


Features
User Authentication (Login / Signup with secure sessions)
 Trading Dashboard (Zerodha-style interface)
 Stock Price Visualization (charts & trends)
 Portfolio Tracking (holdings, profit/loss)
 Buy & Sell Functionality (simulated trading)
 Order History Management
 Responsive UI (works on all devices)
